# Autohotkey-startup-scripts
 Autohotkey scripts to help with my working enviroment and hotkey management

### spanish-n.ahk
Script to handle ñ and Ñ in non spanish keyboards by just typing nhx and NHX (nh is a similar sound to ñ)

### delayed-startup.ahk
script to handle starting up various programs with delay and context awareness and reduce the start up times.
 e.g start steam 30 seconds after the start of windows, or start teamviewer only if nobody has touched the mouse in the first minute of being started up.

### ethminer.ahk
Script to set correct gpu profile before launching ethminer and then reverting back to normal overclocked profile when closed

### background.ahk
Script with a set of hotkeys, hotstings and other personal QoL improvements for windows
